# Flask Demo Application

A comprehensive Flask web application demonstrating best practices and various features of the Flask framework.

## Features

- **Clean Architecture**: Organized project structure following Flask best practices
- **Template Inheritance**: Reusable templates with Jinja2
- **Form Handling**: WTForms with validation and CSRF protection
- **REST API**: Sample API endpoints with JSON responses
- **Error Handling**: Custom 404 and 500 error pages
- **Flash Messages**: User feedback with category support
- **Bootstrap 5**: Responsive, mobile-first design
- **AJAX Integration**: Dynamic data fetching from API

## Technologies Used

- **Backend**: Python 3.9+, Flask 2.3.3
- **Frontend**: HTML5, CSS3, JavaScript, Bootstrap 5
- **Forms**: Flask-WTF, WTForms
- **Template Engine**: Jinja2
